# Powerfull Gym 2 Website

Bu proje, bir spor salonu web sitesi için geliştirilen tek sayfalık responsive bir tasarımdır. HTML, CSS ve JavaScript kullanılarak oluşturulmuştur.

## Özellikler

- **Responsive Tasarım:** Mobil uyumlu (özellikle 576px ekranlar için optimize edildi).
- **BMI Hesaplayıcı:** Kullanıcının girdiği boy ve kiloya göre BMI değerini hesaplar ve görsel üzerinde gösterir.
- **Dinamik İçerik:** "Our Classes" bölümünde butonlara tıklandığında içerik değişir.
- **Hover Efektleri:** Eğitmen kartlarında özel CSS hover animasyonları.
- **Navbar:** Scroll esnasında arka plan rengi değişen sabit navigasyon.

## Kullanılan Teknolojiler

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6)
- Google Fonts & FontAwesome

## Kurulum ve Çalıştırma

1. Repoyu klonlayın: `git clone https://github.com/kullaniciadi/powerfull-gym.git`
2. Klasöre gidin ve `index.html` dosyasını tarayıcınızda açın.
